# Scripts Directory

This directory contains utility scripts for testing, analysis, and development.

## Available Scripts

### Testing Scripts
- `show_results.py` - Run a complete risk assessment and display results
- `test_setup.py` - Check dependencies, project structure, and API key configuration
- `quick_test.py` - Quick validation test for Pydantic schemas

### Analysis Scripts
- `cost_estimator.py` - Estimate the cost per operation (USD and CNY)
- `time_analysis.py` - Analyze system execution time based on historical data
- `measure_time.py` - Measure actual execution time of a risk assessment
- `analyze_pass_rate.py` - Analyze revision patterns and pass rates

## Usage

All scripts can be run directly:
```bash
python scripts/show_results.py
python scripts/cost_estimator.py
python scripts/time_analysis.py
```
